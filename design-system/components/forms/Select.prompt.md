Native `<select>` wrapped in the shared Field shell (label + hint + error) with a terracotta focus ring and chevron.

```jsx
<Select label="Categoria" defaultValue="alimentari">
  <option value="alimentari">Alimentari</option>
  <option value="abbigliamento">Abbigliamento</option>
  <option value="casa">Casa & Cucina</option>
</Select>
```

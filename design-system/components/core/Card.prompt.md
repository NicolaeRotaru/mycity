Standard surface container for any boxed content. Warm shadows on cream, neutral border by default.

```jsx
<Card variant="elevated" padding="lg" as="article">…</Card>
<Card variant="bordered" hover>Product / store card body</Card>
```

Variants: `bordered` (white + cream border, default), `elevated` (adds warm shadow), `flat` (cream-50, no shadow). Padding `none`/`sm`/`md`/`lg`. Set `hover` for the universal -3px lift + warm-lg shadow.

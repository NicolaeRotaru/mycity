Inline promo / status pill used across product cards, the product page and checkout. One source of truth for the marketplace's micro-labels.

```jsx
<Badge variant="discount">-30%</Badge>
<Badge variant="new">Nuovo</Badge>
<Badge variant="free" icon="truck">Sped. gratis</Badge>
<Badge variant="lowstock" icon="flame">Ultimi 2</Badge>
<Badge variant="local">Negozio locale</Badge>
```

Variants: `discount` (wine, uppercase), `new` (olive), `soldout` (ink), `lowstock` (wine), `free`/`cod` (soft olive), `local` (soft terracotta), `urgency` (mustard). Sizes `sm` (default) / `md`.
